Add images to your pages with beautiful hover effects and captions.


== Description ==

Hover Effect is a visual composer extesion which will allow you to add images to your post and pages with caption and attractive hover effects. It also allow you to open your content in lightbox which give an ease to user to read complete content.

Note: This plugin requires Visual Composer Plugin.

<h2>Features</h2>
<ul>
	<li> 20 Unique Hover Effects (More to Come)  </li>
	<li> Title and Description </li>
	<li> External Linking  </li>
	<li> Lighbox Enabled  </li>
	<li> Unlimited Layouts  </li>
</ul>


== Installation ==
1. Go to plugins in your dashboard and select 'add new'
2. Search for 'Hover Effects With Lightbox - VC Extension', install and activate it
3. Go to post and page and select element naming with "Hover Effects With Lightbox - VC Extension".
4. Choose desired options and save settings.
5. View page or post.
